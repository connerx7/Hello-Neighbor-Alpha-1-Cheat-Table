# Hello-Neighbor-Alpha-1-Cheat-Table
A Cheat Engine table for Hello Neighbor Alpha 1 with teleports, noclip, fly, and more.

## Keybinds

- `F1` — Teleport to player house
- `F2` — Fly
- `F3` — Teleport to lockpick
- `F4` — Teleport to basement
- `F5` — Teleport to cutscene house
- `Arrow Keys` — Noclip
- `F6 + Throw Flashlight` — Break Living Room Windows
- `F7` — Teleport to circus
